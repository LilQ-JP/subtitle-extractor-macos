## はじめに

「字幕抽出ツール」は、ゲーム動画から字幕テキストを自動抽出し、SRTファイルとして保存できるデスクトップアプリケーションです。

### 主な機能

- **OCR字幕抽出**: meikiOCRを使用した高精度な日本語字幕認識
- **スクロール字幕検出**: ゲームでよくあるスクロール表示の字幕を正確に検出
- **AI翻訳**: Ollamaを使用したローカルAI翻訳（日本語→英語など）
- **字幕編集**: 抽出した字幕の手動編集・修正
- **SRT出力**: 業界標準のSRT形式で字幕ファイルを保存

## インストール方法

### 1. Pythonのインストール

Pythonがインストールされていない場合は、[python.org](https://www.python.org/downloads/)からダウンロードしてインストールしてください。

インストール時に「Add Python to PATH」にチェックを入れてください。

### 2. 依存パッケージのインストール

コマンドプロンプト（Windows）またはターミナル（Mac/Linux）を開き、以下のコマンドを実行します：

```bash
cd 字幕抽出ツールのフォルダ
pip install -r requirements.txt
```

### 3. meikiOCRのインストール

```bash
pip install meikiocr
```

### 4. FFmpegのインストール（音声再生用）

**Windows:**
1. [FFmpeg公式サイト](https://ffmpeg.org/download.html)からダウンロード
2. 解凍して任意のフォルダに配置
3. 環境変数PATHに追加

**Mac:**
```bash
brew install ffmpeg
```

**Linux:**
```bash
sudo apt install ffmpeg
```

### 5. アプリケーションの起動

```bash
python main_app.py
```

## 基本的な使い方

### ステップ1: 動画を開く

1. 「動画を開く」ボタンをクリック
2. 字幕を抽出したい動画ファイルを選択
3. 動画がプレビュー画面に表示されます

### ステップ2: 字幕領域を選択（任意）

字幕が表示される領域が限定されている場合：

1. 動画プレビュー上でマウスをドラッグ
2. 字幕が表示される領域を囲みます
3. 緑色の枠が表示されれば設定完了

**ヒント**: 字幕領域を設定すると、ゲームUIの誤認識を防げます。

### ステップ3: 字幕を抽出

1. 「字幕を抽出」ボタンをクリック
2. 進捗バーで処理状況を確認
3. 完了すると字幕リストに結果が表示されます

### ステップ4: 字幕を編集（任意）

1. 字幕リストから編集したい項目をクリック
2. 下部の編集パネルでテキストを修正
3. 「変更を適用」ボタンで保存

### ステップ5: SRTファイルを保存

1. 「SRT保存」ボタンをクリック
2. 保存先とファイル名を指定
3. 完了

## AI翻訳機能の設定

AI翻訳を使用するには、**Ollama**のインストールが必要です。

### Ollamaのインストール

#### Windows

1. [Ollama公式サイト](https://ollama.com/download)にアクセス
2. 「Download for Windows」をクリック
3. ダウンロードしたインストーラーを実行
4. インストール完了後、コマンドプロンプトで以下を実行：

```bash
ollama pull llama3.2:3b
```

#### Mac

```bash
# Homebrewでインストール
brew install ollama

# モデルをダウンロード
ollama pull llama3.2:3b
```

#### Linux

```bash
# インストール
curl -fsSL https://ollama.com/install.sh | sh

# モデルをダウンロード
ollama pull llama3.2:3b
```

### 翻訳の実行

1. 字幕を抽出
2. 「AI翻訳」ボタンをクリック
3. 翻訳完了後、「翻訳SRT保存」で保存

### 自動翻訳の設定

1. 「ツール」→「詳細設定」を開く
2. 「翻訳設定」タブを選択
3. 「抽出後に自動で翻訳する」にチェック
4. 「OK」をクリック

## 詳細設定

「ツール」→「詳細設定」から各種設定を変更できます。

### 抽出設定

| 設定項目 | 説明 | 推奨値 |
|----------|------|--------|
| サンプリングFPS | 1秒あたりの解析フレーム数 | 2.0 |
| スクロール字幕検出 | スクロール表示の字幕を検出 | ON |
| 類似度閾値 | 同一字幕と判定する類似度 | 0.6 |
| 安定フレーム数 | 字幕確定に必要なフレーム数 | 2 |
| 最小字幕長 | 字幕の最小表示時間（秒） | 0.5 |
| 最大字幕長 | 字幕の最大表示時間（秒） | 10.0 |

### 翻訳設定

| 設定項目 | 説明 |
|----------|------|
| 抽出後に自動で翻訳する | ONにすると抽出完了後に自動翻訳 |
| 翻訳元言語 | 元の字幕の言語（ja, en, zh, ko） |
| 翻訳先言語 | 翻訳後の言語 |
| Ollamaモデル | 使用するAIモデル |

### 表示設定

| 設定項目 | 説明 | 推奨値 |
|----------|------|--------|
| 字幕フォントサイズ | プレビュー時のフォントサイズ | 24 |
| 字幕背景透明度 | 背景の透明度（0-255） | 180 |

## トラブルシューティング

### 「meikiocr」が見つからない

```bash
pip install meikiocr
```

### 音声が再生されない

FFmpegがインストールされているか確認してください：

```bash
ffmpeg -version
```

### AI翻訳が動作しない

1. Ollamaが起動しているか確認：
```bash
ollama list
```

2. モデルがダウンロードされているか確認：
```bash
ollama pull llama3.2:3b
```

### 字幕が正しく認識されない

- 字幕領域を正確に設定してください
- サンプリングFPSを上げてみてください（3.0〜5.0）
- 動画の解像度が低い場合は認識精度が下がります

### アプリが重い・フリーズする

- 動画の解像度が高すぎる場合は、事前に720pなどに変換してください
- メモリ不足の可能性があります。他のアプリを閉じてください
