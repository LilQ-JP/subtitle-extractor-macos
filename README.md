# 字幕抽出・編集ツール v3.0

日本語ゲーム動画から字幕を自動抽出し、編集・翻訳できるデスクトップアプリケーションです。

## 主な機能

### OCRエンジン（6種類対応）
| エンジン | 精度 | 速度 | コスト | 推奨用途 |
|---------|------|------|--------|----------|
| **EasyOCR** | 中 | 速い | 無料 | 一般的な字幕 |
| **manga-ocr** | 高 | 普通 | 無料 | 漫画・アニメ字幕 |
| **meikiOCR** | 最高 | 普通 | 無料 | ゲーム字幕 |
| **Qwen2.5-VL** | 高 | 遅い | 無料 | 複雑な字幕（Ollama経由） |
| **Gemini Vision** | 高 | 速い | 有料 | 高精度が必要な場合 |
| **GPT-4 Vision** | 最高 | 速い | 有料 | 最高精度が必要な場合 |

### スクロール字幕検出
ゲームの横スクロール字幕に対応。スクロール中の部分テキストを自動除外し、完全表示された字幕のみをキャプチャします。

### 動画プレビュー
- 字幕オーバーレイ表示（ON/OFF切り替え可能）
- 音声付き再生
- シークバーで任意の位置にジャンプ
- 字幕領域のマウスドラッグ選択

### 字幕編集
- 字幕リストの表示・編集
- 開始/終了時間の調整
- テキストの編集
- 翻訳テキストの追加
- SRTファイルの読み込み/保存

## インストール

### 必要条件
- Python 3.9以上
- FFmpeg（音声再生に必要）

### 依存関係のインストール

```bash
pip install -r requirements.txt
```

### オプション: Ollama（ローカルAI）

Qwen2.5-VLやAI校正/翻訳機能を使用する場合：

```bash
# Ollamaをインストール
curl -fsSL https://ollama.com/install.sh | sh

# Qwen2.5-VLモデルをダウンロード
ollama pull qwen2.5vl:7b

# 翻訳/校正用モデル
ollama pull llama3.2:3b
```

### オプション: meikiOCR

```bash
pip install meikiocr
```

## 使い方

### 起動

```bash
python main_app.py
```

### 基本的な流れ

1. **動画を選択** - 「参照...」ボタンで動画ファイルを選択
2. **字幕領域を選択**（オプション）- 動画プレビュー上でドラッグして字幕が表示される領域を指定
3. **OCRエンジンを選択** - 用途に応じてエンジンを選択
4. **設定を調整** - サンプリングFPS、スクロール検出などを設定
5. **字幕を抽出** - 「字幕を抽出」ボタンをクリック
6. **字幕を編集** - 必要に応じて字幕リストで編集
7. **保存** - SRTファイルとして保存

### キーボードショートカット

- `Space` - 再生/一時停止
- `←` - 5秒戻る
- `→` - 5秒進む

## 出力形式

### SRTファイル
標準的なSRT形式で字幕を保存できます。

```
1
00:00:01,000 --> 00:00:03,500
こんにちは、世界！

2
00:00:04,000 --> 00:00:06,000
これはテスト字幕です。
```

## トラブルシューティング

### 音声が再生されない
- FFmpegがインストールされているか確認してください
- `ffplay`コマンドが使用可能か確認してください

### OCRの精度が低い
- 字幕領域を正確に選択してください
- サンプリングFPSを上げてみてください
- より高精度なOCRエンジンを試してください

### Qwen2.5-VLが動作しない
- Ollamaが起動しているか確認してください
- `ollama list`でモデルがインストールされているか確認してください
